# OS Scheduling Simulator

A Java Swing-based desktop application developed as an Operating Systems course project. This application simulates various CPU Scheduling Algorithms and helps users understand process scheduling by calculating waiting time, turnaround time, and visualizing process execution through a Gantt Chart.

## Features

- First Come First Serve (FCFS)
- Shortest Job First (SJF)
- Priority Scheduling
- Round Robin (RR)
- Gantt Chart Visualization
- Process Performance Graph
- Average Waiting Time Calculation
- Average Turnaround Time Calculation
- User-friendly Java Swing GUI

## Technologies Used

- Java
- Java Swing
- NetBeans IDE
- Object-Oriented Programming (OOP)

## How to Run

1. Clone the repository.
2. Open the project in NetBeans IDE.
3. Build the project.
4. Run the application.
5. Select a scheduling algorithm.
6. Enter process details.
7. View the scheduling results, Gantt Chart, and Performance Graph.
